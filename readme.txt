This plugin automatically optimizes and converts images to WebP format upon upload.

== Description ==
Image Optimization and WebP Conversion is a simple and effective plugin for automatically optimizing and converting images to WebP format when they are uploaded to your WordPress site.

The plugin allows you to:
* Automatically convert uploaded images to WebP format.
* Resize images if they are larger than a specified dimension.
* Choose the quality of the converted images.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/image-optimization-webp-conversion` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Image Optimization screen to configure the plugin.

== Frequently Asked Questions ==
= Do I need any PHP extension to run this plugin? =
Yes, you can trigger this function through a button in the plugin's settings panel.

If you plan to run this code on a server with PHP 7.2, you need to make sure that the PHP installation includes the GD extension and it was compiled with WebP support.
